# toxcore-ios

Cocoapods wrapper for [toxcore](https://github.com/irungentoo/toxcore).

## Installation

toxcore-ios is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

    pod "toxcore-ios"
